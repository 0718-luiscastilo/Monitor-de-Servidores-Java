# 🖥️ Monitor de Servidores

## 📌 Descripción

Monitor de Servidores es una aplicación de consola desarrollada en Java que permite registrar información de múltiples servidores y generar un reporte de utilización de recursos.

El sistema analiza el uso de CPU y memoria RAM para clasificar automáticamente el estado de cada servidor y generar estadísticas generales.

---

## 🚀 Funcionalidades

### Registro de servidores
- Nombre del servidor
- Dirección IP
- Uso de CPU (%)
- Uso de RAM (%)

### Análisis automático
- Estado Normal
- Estado Alerta
- Estado Crítico

### Reportes
- Listado completo de servidores
- Promedio de uso de CPU
- Promedio de uso de RAM
- Servidor con mayor uso de CPU
- Servidor con mayor uso de RAM
